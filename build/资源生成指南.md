# 安装包资源生成指南

由于 Windows 系统自带的 `convert.exe` 命令会与 ImageMagick 冲突，请按以下方式生成资源：

## 方法一：使用在线工具（推荐，最简单）

### 1. 生成 icon.ico

访问：https://convertio.co/zh/png-ico/

- 上传 `resources/yuanjiao-Livo.png`
- 选择输出尺寸：256x256
- 下载并保存为 `resources/icon.ico`

### 2. 生成 installerSidebar.bmp

访问：https://convertio.co/zh/png-bmp/

- 上传 `resources/yuanjiao-Livo.png`
- 调整尺寸为：164x314
- 下载并保存为 `resources/installerSidebar.bmp`

### 3. 生成 installerHeader.bmp

访问：https://convertio.co/zh/png-bmp/

- 上传 `resources/yuanjiao-Livo.png`
- 调整尺寸为：150x57
- 下载并保存为 `resources/installerHeader.bmp`

## 方法二：使用 PowerShell + ImageMagick

如果已安装 ImageMagick，在 PowerShell 中运行：

```powershell
cd D:\project\Livo-project\Livo\resources

# 使用完整路径调用 magick
& "C:\Program Files\ImageMagick-7.1.1-Q16\magick.exe" convert yuanjiao-Livo.png -resize 256x256 -define icon:auto-resize=256,128,64,48,32,16 icon.ico

& "C:\Program Files\ImageMagick-7.1.1-Q16\magick.exe" convert yuanjiao-Livo.png -resize 150x57 -gravity center -background white -extent 150x57 installerHeader.bmp

& "C:\Program Files\ImageMagick-7.1.1-Q16\magick.exe" convert yuanjiao-Livo.png -resize 164x314 -gravity center -background white -extent 164x314 installerSidebar.bmp
```

（请根据实际 ImageMagick 安装路径调整）

## 方法三：使用 Python PIL

```python
from PIL import Image

# 打开原图
img = Image.open('yuanjiao-Livo.png')

# 生成 icon.ico
img.resize((256, 256)).save('icon.ico')

# 生成 installerSidebar.bmp
sidebar = Image.new('RGB', (164, 314), 'white')
img_resized = img.resize((164, 314))
sidebar.paste(img_resized, (0, 0))
sidebar.save('installerSidebar.bmp')

# 生成 installerHeader.bmp
header = Image.new('RGB', (150, 57), 'white')
img_resized = img.resize((150, 57))
header.paste(img_resized, (0, 0))
header.save('installerHeader.bmp')
```

## 完成后

生成三个文件后，运行：

```bash
cd D:\project\Livo-project\Livo
pnpm run build:win
```

安装包将生成在 `dist/` 目录。
